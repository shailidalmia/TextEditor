Functionalities Implemented

  • Editable space using contenteditable
  • Pressing enter key starts a new paragraph
  • Formatting Toolbar
    Double clicking a word or selecting some text displays a toolbar above the selection which has buttons for foramtting the text to make it - 1) Bold 2) Italics 3) Underline 4) Change font color to red.
  • On the fly link creation!
    When the user types some text between <a> tags, the broswer prompts for a link from the user as soon as he/she finishes typing the closing angular bracket. The link is then shifted to the bottom of the page and it redirects to the url given as an input by the user.
  • Randomize button!
    On clicking this button, all four letter words from the editable text area area are replaced by a random english word sourced from the website http://randomword.setgetgo.com. Requires internet connection. A little buggy though :[
